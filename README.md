# Project Dream

## Overview

Project Dream is a solo-developed 2D platformer built in Unreal Engine 5 that focuses on responsive player movement, enemy AI, and handcrafted gameplay interactions. The project was created to strengthen my understanding of gameplay programming, level design, and Blueprint scripting while developing a complete gameplay experience from the ground up.

Players navigate platforming challenges, avoid enemy encounters, and progress through levels using responsive movement mechanics and interactive gameplay elements.

## Technical Challenges

One of the biggest challenges during development was creating responsive player movement that felt smooth and consistent while interacting with Unreal Engine's collision system. Careful tuning of movement values and collision responses was necessary to produce controls that felt natural during platforming.

Another challenge involved implementing enemy AI and gameplay interactions using Blueprint scripting while maintaining an organized project structure. Developing these systems independently strengthened my understanding of gameplay architecture and debugging techniques.

**Completed**
