# Project Dream

## Overview

Project Dream is a solo-developed 2D platformer built in Unreal Engine 5 that focuses on responsive player movement, platforming mechanics, enemy AI, and handcrafted gameplay interactions. The project was created to strengthen my understanding of gameplay programming, Blueprint scripting, and level design while developing a complete gameplay experience from the ground up.

Players navigate platforming challenges using movement abilities, collect items throughout each level, defeat enemies by jumping on their heads, and progress through handcrafted environments filled with interactive obstacles.

## Technical Challenges

One of the biggest challenges during development was creating responsive platforming mechanics that felt smooth and reliable. Balancing player acceleration, jump height, gravity, collision detection, and double jump timing required continuous iteration to create satisfying movement.

Another challenge involved developing enemy AI and gameplay interactions entirely through Blueprint scripting while keeping the project organized and maintainable. Creating moving platforms, collectible systems, and enemy behaviors strengthened my understanding of gameplay architecture and debugging within Unreal Engine.
